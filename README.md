# 学校运动会管理系统 (C++)

本项目是一个基于C++实现的学校运动会管理系统，旨在提供运动会相关的各项管理功能。

## 项目目标

- 实现参赛单位、运动员、比赛项目的管理。
- 实现运动员报名登记，并满足相关限制条件。
- 实现计分规则的灵活设置。
- 自动生成秩序册（高级功能，逐步完善）。
- 实现比赛成绩的录入、查询、统计和汇总。
- 提供数据备份与恢复功能。

## 当前阶段

项目处于初期开发阶段，已完成核心数据结构和部分基础功能的框架搭建。

SchoolSportsMeetManagementSystem/

## 核心模块说明

- **Unit**: 参赛单位类
- **Athlete**: 运动员类
- **CompetitionEvent**: 比赛项目类
- **ScoreRule**: 计分规则类
- **SystemSettings**: 系统全局设置与数据管理中心
- **Registration**: 报名管理类
- **Schedule**: 赛程与秩序册管理类
- **Result**: 比赛成绩类
- **DataManager**: 数据持久化管理类
- **Constants**: 全局常量定义

## 如何编译与运行 (示例)

假设使用 g++ 编译器：

1.  **编译所有源文件**:
    ```bash
    g++ -std=c++20 -Wall -Wextra src/*.cpp -Iinclude -o sports_meet_system
    ```
    (-std=c++20 是因为使用了 `std::atomic` 和一些现代C++特性, `-Iinclude` 指定头文件搜索路径)

2.  **运行程序**:
    ```bash
    ./sports_meet_system
    ```
